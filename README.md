# AI Job Portal

An AI-powered full-stack job portal built with React, FastAPI, and PostgreSQL — enhanced with agentic AI features for resume analysis, job matching, and interview preparation.

## 🚀 Project Overview

This project is being built incrementally as a learning and portfolio project, combining traditional full-stack development with modern AI integration and agentic workflows.

## 🛠️ Tech Stack

**Backend:** FastAPI (Python), PostgreSQL, SQLAlchemy, JWT Authentication
**Frontend:** React, JavaScript
**AI Integration:** OpenAI / Gemini APIs, Prompt Engineering, RAG, Vector Databases
**Agentic AI:** LangChain, LangGraph
**Tools:** Git, GitHub, Docker (planned)

## ✨ Features

### Core Features (In Progress)
- User authentication (JWT) for candidates and recruiters
- Candidate dashboard
- Recruiter dashboard
- Job posting, search, and application system
- Resume upload

### AI Features (Planned)
- 🤖 Resume Analyzer with ATS scoring
- ✍️ AI Cover Letter Generator
- ❓ AI Interview Question Generator
- 🎯 AI-powered job matching
- 🧠 Agentic workflow: resume → job match → cover letter → interview prep → study plan
